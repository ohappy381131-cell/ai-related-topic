## 1. Diffusion 모델이란

Diffusion 모델은 2020년 이후 이미지 생성 분야에서 GAN을 빠르게 대체하며 현재 가장 강력한 생성 모델로 자리 잡은 방법입니다. Stable Diffusion, DALL-E 2, Midjourney, Imagen 모두 Diffusion 모델 기반입니다.

Diffusion 모델의 핵심 아이디어는 물리학의 확산(diffusion) 현상에서 영감을 받았습니다. 잉크 한 방울이 물에 퍼져나가는 것처럼, 이미지에 노이즈를 조금씩 추가하면 결국 완전한 랜덤 노이즈가 됩니다. Diffusion 모델은 이 과정을 **역방향으로 학습**합니다. 노이즈에서 시작해 조금씩 노이즈를 제거하면서 의미 있는 이미지를 복원하는 것입니다.

---

## 2. 두 가지 핵심 과정: 순방향과 역방향

Diffusion 모델은 두 가지 과정으로 구성됩니다.

### 순방향 과정 (Forward Process): 노이즈 추가

학습 데이터(원본 이미지)에 단계적으로 가우시안 노이즈를 추가합니다. T 스텝에 걸쳐 노이즈를 조금씩 더하면, 마지막 스텝에서는 원본 이미지 정보가 완전히 사라진 순수한 가우시안 노이즈가 됩니다.

```
순방향 과정 (학습 시 사용, 고정된 과정):

원본 이미지 x₀
    ↓ 노이즈 추가 (step 1)
약간 노이즈 낀 이미지 x₁
    ↓ 노이즈 추가 (step 2)
더 노이즈 낀 이미지 x₂
    ↓
    ...
    ↓ 노이즈 추가 (step T)
완전한 가우시안 노이즈 xT
```

이 과정은 학습 가능한 파라미터가 없습니다. 수학적으로 정의된 고정된 과정입니다.

### 역방향 과정 (Reverse Process): 노이즈 제거

역방향 과정이 실제로 학습하는 부분입니다. 노이즈가 낀 이미지 xₜ를 입력받아 이전 스텝의 이미지 xₜ₋₁을 예측하는 신경망을 학습합니다. 구체적으로는 각 스텝에서 추가된 노이즈가 무엇인지를 예측하는 방향으로 학습합니다.

```
역방향 과정 (학습 및 생성 시 사용):

완전한 가우시안 노이즈 xT
    ↓ 노이즈 예측 및 제거 (step T)
덜 노이즈 낀 이미지 xT₋₁
    ↓ 노이즈 예측 및 제거 (step T-1)
    ...
    ↓ 노이즈 예측 및 제거 (step 1)
복원된 이미지 x₀
```

역방향 과정의 각 스텝에서 노이즈를 예측하는 모델로 U-Net 구조가 주로 사용됩니다. U-Net의 인코더-디코더 구조와 스킵 연결이 노이즈 예측에 적합하기 때문입니다.

## 3. 이미지 생성에서의 활용

### 무조건부 이미지 생성 (Unconditional Generation)

가장 기본적인 형태로, 순수한 가우시안 노이즈에서 시작해 역방향 과정을 T 스텝 반복하면 새로운 이미지가 생성됩니다. 어떤 조건도 없이 학습 데이터의 분포를 따르는 새로운 이미지를 무작위로 생성합니다.

### 텍스트 조건부 생성 (Text-to-Image)

현재 가장 주목받는 활용 방식입니다. 텍스트 프롬프트를 조건으로 이미지를 생성합니다. CLIP 같은 텍스트 인코더로 텍스트를 벡터로 변환하고, 이 벡터를 노이즈 예측 과정에 주입해 텍스트 내용에 맞는 이미지를 생성합니다. Stable Diffusion, DALL-E 2, Imagen이 이 방식을 사용합니다.

```
"우주복을 입은 고양이가 달에서 기타를 치는 장면"
    ↓ 텍스트 인코더 (CLIP 등)
텍스트 임베딩 벡터
    ↓ 역방향 Diffusion 과정에 조건으로 주입
가우시안 노이즈 → ... → 생성된 이미지
```

### 이미지 조건부 생성 (Image-to-Image)

기존 이미지를 입력으로 받아 변형하거나 스타일을 바꾸는 방식입니다. 원본 이미지에 일정량의 노이즈를 추가한 뒤, 새로운 조건으로 역방향 과정을 진행합니다. 낮 사진을 밤 사진으로 변환하거나, 스케치를 실제 이미지로 변환하는 데 활용됩니다.

### 이미지 인페인팅 (Inpainting)

이미지의 특정 영역을 마스킹한 뒤, 마스킹된 영역만 역방향 과정으로 채우는 방식입니다. 이미지에서 원하지 않는 객체를 자연스럽게 제거하거나, 손상된 이미지를 복원하는 데 사용됩니다. 역방향 과정에서 마스킹되지 않은 영역은 원본을 유지하면서, 마스킹된 영역만 주변 맥락에 맞게 생성합니다.

### Latent Diffusion Model (잠재 공간 Diffusion)

Stable Diffusion이 사용하는 방식입니다. 픽셀 공간에서 직접 Diffusion을 수행하는 대신, 오토인코더로 이미지를 저차원 잠재 공간(latent space)으로 압축한 뒤 잠재 공간에서 Diffusion을 수행합니다. 계산 비용이 크게 줄어들고, 고해상도 이미지 생성이 가능해집니다.

```
픽셀 공간 Diffusion:
이미지 (512×512×3) → Diffusion → 이미지 (512×512×3)
→ 계산 비용 매우 큼

Latent Diffusion (Stable Diffusion):
이미지 (512×512×3) → [인코더] → 잠재 벡터 (64×64×4)
→ 잠재 공간에서 Diffusion 수행 (훨씬 빠름)
→ [디코더] → 이미지 (512×512×3)
```

## 4. Diffusion 모델의 장점

### GAN 대비 학습 안정성

GAN의 가장 큰 문제는 생성자와 판별자의 균형 잡힌 학습이 어렵다는 것입니다. 모드 붕괴, 학습 불균형 같은 문제가 빈번합니다. Diffusion 모델은 단일 신경망이 노이즈를 예측하는 방식으로 학습하므로, 두 네트워크의 균형을 맞출 필요가 없습니다. 학습이 안정적이고 재현성이 높습니다.

### 높은 이미지 품질과 다양성

Diffusion 모델은 T 스텝에 걸쳐 점진적으로 이미지를 생성하므로, 각 스텝에서 세밀한 조정이 가능합니다. 결과적으로 GAN보다 더 높은 품질의 이미지를 생성하고, 모드 붕괴 없이 다양한 이미지를 생성할 수 있습니다. FID(Fréchet Inception Distance) 같은 이미지 품질 지표에서 GAN을 능가하는 결과를 보여줍니다.

### 명시적 확률 모델

Diffusion 모델은 확률론적 토대가 명확합니다. 순방향 과정이 수학적으로 엄밀하게 정의되어 있고, 역방향 과정도 베이즈 추론으로 유도됩니다. 이 덕분에 모델의 동작을 이론적으로 분석하고, 샘플링 과정을 제어하기 쉽습니다.

### 유연한 조건부 생성

텍스트, 이미지, 클래스 레이블, 깊이 맵, 스케치 등 다양한 형태의 조건을 역방향 과정에 주입할 수 있습니다. Classifier-Free Guidance 기법을 통해 조건의 강도를 조절하면서 생성 결과를 세밀하게 제어할 수 있습니다. ControlNet처럼 추가 구조를 붙여 포즈, 엣지, 깊이 등 정밀한 조건을 부여하는 것도 가능합니다.

### 편집 가능성

Diffusion 모델은 이미지 편집에 자연스럽게 적용됩니다. 역방향 과정의 특정 스텝에서 개입해 생성 방향을 바꾸거나, 이미지의 일부 영역만 선택적으로 수정하는 것이 가능합니다. DDIM Inversion처럼 기존 이미지를 노이즈로 변환했다가 다시 조건을 바꿔 생성하는 이미지 편집 기법도 Diffusion 모델의 구조를 활용합니다.

---

## 5. Diffusion 모델의 한계

강력한 장점에도 불구하고 Diffusion 모델에는 중요한 한계가 있습니다.

가장 큰 단점은 **생성 속도**입니다. 역방향 과정에서 T 스텝(보통 수백~수천 스텝)을 반복해야 하므로 이미지 하나를 생성하는 데 상당한 시간이 걸립니다. GAN은 단 한 번의 순전파로 이미지를 생성하는 것과 대조적입니다. 이를 해결하기 위해 DDIM, DPM-Solver 같은 빠른 샘플링 방법이 개발되어 스텝 수를 20~50개로 줄였지만, 여전히 GAN보다 느립니다.

또한 Latent Diffusion 모델은 오토인코더를 통한 압축과 복원 과정에서 일부 세밀한 디테일이 손실될 수 있습니다.

| **핵심 원리** | 노이즈 추가(순방향) → 노이즈 제거 학습(역방향) |
| **주요 활용** | 텍스트-이미지 생성, 이미지 편집, 인페인팅, 스타일 변환 |
| **GAN 대비 장점** | 학습 안정성, 높은 품질과 다양성, 유연한 조건부 생성 |
| **주요 한계** | 생성 속도 느림 |
| **대표 모델** | Stable Diffusion, DALL-E 2, Imagen, Midjourney |

Diffusion 모델은 이미지 생성 분야의 패러다임을 바꿨습니다. GAN이 지배하던 시대에서 Diffusion 모델이 주류가 된 것은 단순히 더 좋은 성능 때문만이 아닙니다. 학습의 안정성, 조건부 생성의 유연성, 이론적 토대의 견고함이 결합된 결과입니다. 생성 속도 문제는 지속적인 연구로 빠르게 개선되고 있으며, 현재 가장 활발하게 발전하는 딥러닝 분야 중 하나입니다.
