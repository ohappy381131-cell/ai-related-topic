## 1. GAN이란: 경쟁으로 학습하는 생성 모델

GAN(Generative Adversarial Network, 생성적 적대 신경망)은 2014년 Ian Goodfellow가 제안한 생성 모델입니다. GAN의 핵심 아이디어는 두 개의 신경망이 서로 경쟁하면서 학습한다는 것입니다. 한쪽은 가짜 데이터를 만들어내고, 다른 쪽은 진짜와 가짜를 구별합니다. 이 경쟁 구조가 두 네트워크를 동시에 발전시킵니다.

비유하자면 위조지폐범(생성자)과 경찰(판별자)의 관계입니다. 위조지폐범은 경찰이 가짜를 잡아낼 수 없을 만큼 정교한 지폐를 만들려 하고, 경찰은 아무리 정교한 위조지폐라도 잡아내려 합니다. 둘이 경쟁할수록 위조지폐는 점점 정교해지고, 경찰의 감별 능력도 점점 높아집니다. 학습이 충분히 진행되면 생성자는 진짜와 구별하기 어려운 데이터를 만들어낼 수 있게 됩니다.

---

## 2. 전체 구조 개요

GAN은 생성자(Generator)와 판별자(Discriminator) 두 개의 신경망으로 구성됩니다.

```
노이즈 z (랜덤 벡터)
    ↓
[생성자 G]
    ↓
가짜 이미지 G(z)
    ↓                    ← 진짜 이미지 x (실제 데이터)
[판별자 D]
    ↓
확률값 D(x) 또는 D(G(z))
→ 1에 가까울수록 "진짜"
→ 0에 가까울수록 "가짜"
```

두 네트워크는 서로 반대의 목표를 가집니다. 생성자는 판별자를 속이려 하고, 판별자는 속지 않으려 합니다. 이 적대적 관계(adversarial relationship)가 GAN이라는 이름의 유래입니다.

---

## 3. 생성자 (Generator)

### 역할과 목표

생성자의 역할은 **진짜처럼 보이는 가짜 데이터를 만드는 것**입니다. 입력으로 무작위 노이즈 벡터 z를 받아, 이것을 실제 데이터와 유사한 출력(이미지, 텍스트, 음성 등)으로 변환합니다. 생성자의 목표는 판별자가 자신이 만든 가짜를 진짜로 착각하게 만드는 것입니다.

수식으로 표현하면, 생성자 G는 노이즈 z를 입력받아 G(z)를 출력합니다. 생성자는 D(G(z))가 1(진짜)에 가깝게 되도록 학습합니다. 판별자가 "진짜"라고 판단하는 데이터를 만들수록 생성자가 잘 학습된 것입니다.

### 구조

생성자는 저차원 노이즈 벡터를 고차원 데이터로 변환하는 구조입니다. 이미지 생성의 경우, 오토인코더의 디코더와 유사하게 작은 특성 맵에서 시작해 전치 합성곱(Transposed Convolution)으로 해상도를 키워가며 이미지를 생성합니다.

```
노이즈 z (100차원 벡터)
    ↓
FC Layer → 4×4×512
    ↓
전치 합성곱 → 8×8×256
    ↓
전치 합성곱 → 16×16×128
    ↓
전치 합성곱 → 32×32×64
    ↓
전치 합성곱 → 64×64×3
    ↓
생성된 이미지 (64×64 RGB)
```

노이즈 z는 잠재 공간(latent space)에서 샘플링됩니다. 잠재 공간의 서로 다른 위치가 서로 다른 특성의 이미지에 대응합니다. z값을 조금씩 변화시키면 생성되는 이미지도 부드럽게 변화합니다.

### 학습 방식

생성자는 판별자의 출력을 통해 간접적으로 학습합니다. 생성자가 직접 진짜 데이터를 보는 것이 아닙니다. 판별자가 생성된 이미지를 얼마나 가짜라고 판단하는지가 생성자의 손실이 됩니다. 판별자가 가짜라고 확신할수록 생성자의 손실이 크고, 진짜라고 착각할수록 손실이 작습니다. 생성자는 이 손실을 줄이는 방향으로 파라미터를 업데이트합니다.

---

## 4. 판별자 (Discriminator)

### 역할과 목표

판별자의 역할은 **입력된 데이터가 진짜인지 가짜인지를 구별하는 것**입니다. 진짜 데이터(실제 데이터셋에서 가져온 것)와 생성자가 만든 가짜 데이터를 입력받아, 각각이 진짜일 확률을 0~1 사이의 값으로 출력합니다. 판별자의 목표는 진짜는 1에 가깝게, 가짜는 0에 가깝게 출력하는 것입니다.

### 구조

판별자는 이진 분류기(binary classifier)입니다. 이미지를 입력받아 합성곱 레이어를 거치면서 특성을 추출하고, 마지막에 sigmoid 활성화 함수를 통해 0~1 사이의 확률값을 출력합니다. 구조적으로 일반 CNN 분류 모델과 유사합니다.

```
이미지 입력 (진짜 또는 가짜)
    ↓
합성곱 → 32×32×64
    ↓
합성곱 → 16×16×128
    ↓
합성곱 → 8×8×256
    ↓
합성곱 → 4×4×512
    ↓
Flatten → FC Layer
    ↓
Sigmoid → 확률값 (0~1)
→ 1: 진짜로 판단
→ 0: 가짜로 판단
```

### 학습 방식

판별자는 두 가지 상황에서 학습합니다. 진짜 데이터를 입력했을 때 출력이 1에 가까워야 하고, 가짜 데이터를 입력했을 때 출력이 0에 가까워야 합니다. 이 두 가지 오차를 합산한 것이 판별자의 손실입니다. 판별자는 이 손실을 줄이는 방향으로 학습합니다.

---

## 5. 두 네트워크의 적대적 학습 과정

생성자와 판별자는 번갈아가며 학습합니다. 한 번의 학습 스텝은 다음 순서로 이루어집니다.

```
Step 1: 판별자 학습
├── 진짜 데이터 x를 판별자에 입력 → D(x) ≈ 1이 되도록 학습
└── 가짜 데이터 G(z)를 판별자에 입력 → D(G(z)) ≈ 0이 되도록 학습

Step 2: 생성자 학습
└── 가짜 데이터 G(z)를 판별자에 입력 → D(G(z)) ≈ 1이 되도록 학습
    (판별자 파라미터는 고정, 생성자 파라미터만 업데이트)
```

이 과정이 반복되면서 두 네트워크가 동시에 발전합니다. 생성자는 더 정교한 가짜를 만들고, 판별자는 더 예리하게 구별합니다.

GAN의 손실 함수는 다음과 같은 minimax 게임으로 표현됩니다.

```
min_G max_D V(D, G) = E[log D(x)] + E[log(1 - D(G(z)))]

판별자는 이 값을 최대화하려 함 (max_D)
생성자는 이 값을 최소화하려 함 (min_G)
```

이상적인 학습 결과는 생성자가 너무 정교한 가짜를 만들어 판별자가 진짜와 가짜를 구별할 수 없게 되는 상태입니다. 이 상태에서 판별자의 출력은 항상 0.5가 됩니다. 50%의 확률로 진짜, 50%의 확률로 가짜라고 판단하는, 즉 완전히 랜덤하게 찍는 수준이 됩니다. 이것이 GAN의 이론적 균형점(Nash Equilibrium)입니다.

---

## 6. 생성자와 판별자의 비교

| | 생성자 (Generator) | 판별자 (Discriminator) |
|---|---|---|
| **입력** | 랜덤 노이즈 벡터 z | 진짜 또는 가짜 이미지 |
| **출력** | 가짜 데이터 G(z) | 진짜일 확률 (0~1) |
| **목표** | 판별자를 속이는 가짜 생성 | 진짜와 가짜 구별 |
| **손실 기준** | D(G(z))가 1에 가까울수록 손실 감소 | D(x)≈1, D(G(z))≈0일수록 손실 감소 |
| **구조** | 디코더형 (차원 확장) | 인코더형 (차원 축소) |
| **학습 후 사용** | 새로운 데이터 생성 | 학습 후 보통 버림 |

---

## 7. GAN의 학습 불안정성과 주요 문제

GAN은 강력하지만 학습이 불안정한 것으로 유명합니다. 두 네트워크가 균형 있게 발전해야 하는데, 한쪽이 너무 앞서가면 문제가 생깁니다.

**모드 붕괴(Mode Collapse)**는 가장 흔한 문제입니다. 생성자가 판별자를 속이는 데 성공한 특정 패턴만 반복적으로 생성하는 현상입니다. 예를 들어 10개 클래스의 이미지를 생성해야 하는데 하나의 클래스만 계속 생성하는 경우입니다. 다양한 데이터를 생성하지 못하고 특정 패턴에 수렴해버립니다.

**학습 불균형** 문제도 있습니다. 판별자가 너무 강해지면 생성자가 어떻게 만들어도 가짜로 판별되어 유용한 기울기를 받지 못합니다. 반대로 판별자가 너무 약하면 품질 낮은 가짜도 통과시켜 생성자가 발전하지 않습니다.

이런 문제를 해결하기 위해 DCGAN, WGAN, StyleGAN, CycleGAN 등 다양한 변형 GAN이 제안되었습니다.

---

## 핵심 요약

GAN은 생성자와 판별자가 서로 경쟁하면서 학습하는 구조입니다. 생성자는 랜덤 노이즈에서 진짜 같은 데이터를 만드는 법을 학습하고, 판별자는 진짜와 가짜를 구별하는 법을 학습합니다. 이 경쟁이 두 네트워크를 동시에 발전시키고, 충분히 학습된 생성자는 실제 데이터와 구별하기 어려운 새로운 데이터를 만들어낼 수 있습니다. 이미지 생성, 데이터 증강, 스타일 변환, 이미지 복원 등 다양한 분야에서 GAN이 활용됩니다.
