# 19. 데이터 증강(Data Augmentation) — 개념과 이미지 증강 기법

본 글에서는 데이터 증강이 무엇인지, 왜 필요한지, 그리고 이미지 데이터에서 주로 사용하는 증강 기법들을 하나의 흐름 속에서 정리합니다. 기하학적 변환부터 고급 증강 기법까지, 각 기법이 어떤 원리로 작동하고 어떤 상황에 적합한지를 구조적으로 이해할 수 있도록 설명합니다.

## 1. 데이터 증강이란

딥러닝 모델의 성능은 데이터의 양과 질에 크게 의존합니다. 그런데 현실에서는 충분한 양의 레이블된 데이터를 확보하는 것이 어렵고 비용이 많이 듭니다. 의료 이미지에 전문의가 레이블을 달거나, 특수 장비로 촬영한 데이터를 수집하는 것은 시간과 비용이 상당합니다.

데이터 증강(Data Augmentation)은 기존 데이터에 다양한 변환을 적용해 인위적으로 데이터의 양을 늘리는 기법입니다. 새로운 데이터를 수집하는 것이 아니라, 이미 가진 데이터를 변형해 모델이 더 다양한 패턴을 학습하도록 만듭니다.

핵심 아이디어는 **변환을 가해도 레이블이 바뀌지 않는 변환만 적용한다**는 것입니다. 고양이 사진을 좌우로 뒤집어도 여전히 고양이입니다. 밝기를 조금 낮춰도 여전히 고양이입니다. 이처럼 레이블을 유지하면서 입력을 다양하게 변형하면, 모델 입장에서는 더 많은 데이터를 본 것과 같은 효과가 납니다.

---

## 2. 데이터 증강이 필요한 이유

데이터 증강의 효과는 단순히 데이터를 늘리는 것 이상입니다. 두 가지 핵심 역할이 있습니다.

**과적합 방지.** 데이터가 적으면 모델이 훈련 이미지를 통째로 암기해버립니다. 특정 조명, 특정 각도, 특정 배경에서 찍힌 이미지만 반복적으로 보면 그 조건에 과적합됩니다. 데이터 증강은 매 에포크마다 조금씩 다른 모습의 이미지를 모델에 보여줌으로써, 모델이 표면적 특성이 아닌 본질적인 패턴을 학습하도록 강제합니다.

**일반화 성능 향상.** 현실 세계의 데이터는 다양한 조건에서 수집됩니다. 스마트폰으로 찍힌 이미지는 기울어질 수 있고, 야외 사진은 조명이 달라집니다. 증강을 통해 모델이 이런 다양한 조건을 미리 경험하게 하면 실제 배포 환경에서의 성능이 향상됩니다.

## 3. 기하학적 변환 기법

기하학적 변환은 이미지의 픽셀 위치를 변경하는 방법입니다. 객체의 크기, 위치, 방향이 달라진 이미지를 생성해 모델이 위치와 방향에 불변한 특성을 학습하게 합니다.

### 수평/수직 뒤집기 (Flip)

이미지를 좌우 또는 상하로 뒤집습니다. 가장 단순하고 효과적인 증강 기법 중 하나입니다. 자연 이미지, 의료 이미지처럼 좌우 대칭이 의미 없는 도메인에서 특히 유용합니다. 단, 상하 뒤집기는 도메인에 따라 주의가 필요합니다. 위성 이미지에서는 적합하지만, 사람 얼굴이나 자동차 이미지에서는 비현실적인 결과가 생길 수 있습니다.

### 회전 (Rotation)

이미지를 특정 각도 범위 내에서 무작위로 회전합니다. 예를 들어 -30도에서 +30도 사이에서 랜덤하게 회전시킵니다. 회전 후 생기는 빈 영역은 0(검정)으로 채우거나 가장 가까운 픽셀값으로 채웁니다. 의료 영상, 위성 이미지, 현미경 이미지처럼 방향이 고정되지 않은 도메인에 효과적입니다. 반면 텍스트 인식이나 자동차 번호판 인식처럼 방향이 중요한 도메인에서는 적합하지 않습니다.

### 크롭 (Crop)

이미지의 일부 영역만 잘라내는 방법입니다. 랜덤 크롭(Random Crop)은 무작위 위치에서 고정 크기로 잘라내고, 중앙 크롭(Center Crop)은 이미지 중앙을 기준으로 잘라냅니다. 크롭은 객체가 이미지의 어느 위치에 있어도 인식할 수 있도록 모델을 훈련시키는 효과가 있습니다. 또한 리사이징 과정에서 생기는 왜곡을 피하는 방법으로도 활용됩니다.

### 이동 (Translation)

이미지를 수평 또는 수직 방향으로 일정 픽셀만큼 이동합니다. 이동 후 생기는 빈 영역은 0이나 반사된 픽셀로 채웁니다. 객체가 이미지 프레임의 가장자리에 위치하는 경우를 학습 데이터에 포함시키는 효과가 있습니다.

### 전단 변환 (Shear)

이미지를 특정 축 방향으로 기울이는 변환입니다. 직사각형이 평행사변형처럼 변형됩니다. 카메라 각도나 촬영 조건의 변화를 시뮬레이션하는 데 사용됩니다.

## 4. 색상 변환 기법

색상 변환은 이미지의 픽셀값(밝기, 색상, 대비 등)을 변경하는 방법입니다. 조명 조건, 카메라 설정, 계절 변화 등 현실에서 발생하는 색상 변동을 시뮬레이션합니다.

### 밝기 조절 (Brightness)

이미지 전체의 밝기를 무작위로 높이거나 낮춥니다. 실내외 조명 차이, 낮과 밤의 조명 변화를 시뮬레이션합니다. 자율주행, 보안 카메라처럼 다양한 조명 환경에서 동작해야 하는 모델 훈련에 특히 중요합니다.

### 대비 조절 (Contrast)

이미지의 밝은 부분과 어두운 부분 사이의 차이를 조절합니다. 대비가 높으면 선명해 보이고, 낮으면 뿌옇게 보입니다. 안개가 낀 날씨나 역광 상황을 시뮬레이션하는 데 활용됩니다.

### 채도 조절 (Saturation)

색상의 선명함을 조절합니다. 채도를 낮추면 회색에 가까워지고, 높이면 색상이 더 강렬해집니다. 0으로 낮추면 그레이스케일 이미지가 됩니다. 다양한 카메라 설정이나 계절 변화를 시뮬레이션합니다.

### 색조 조절 (Hue)

색상환에서의 색조를 회전시킵니다. 빨간색이 약간 주황빛이나 보라빛으로 바뀌는 정도의 변환입니다. 과하게 적용하면 비현실적인 색상이 되므로 미세하게 적용하는 것이 일반적입니다.

### Color Jitter

밝기, 대비, 채도, 색조를 동시에 무작위로 변환하는 방법입니다. PyTorch의 `torchvision.transforms.ColorJitter`가 이 네 가지를 한꺼번에 처리합니다. 색상 관련 증강을 한 번에 처리할 수 있어 실무에서 가장 많이 사용됩니다.

## 5. 노이즈 및 블러 기법

### 가우시안 노이즈 (Gaussian Noise)

픽셀값에 정규 분포를 따르는 무작위 노이즈를 추가합니다. 실제 카메라 센서의 노이즈를 시뮬레이션합니다. 모델이 노이즈에 강건하게 학습되어 이상치 탐지나 의료 영상 분석에서 효과적입니다.

### 가우시안 블러 (Gaussian Blur)

이미지를 부드럽게 흐리게 만듭니다. 초점이 맞지 않은 사진이나 빠른 움직임에서 생기는 모션 블러를 시뮬레이션합니다. 고해상도 특성에 과적합하는 것을 방지하는 효과도 있습니다.

---

## 6. 고급 증강 기법

기본 변환을 넘어 더 강력한 증강 효과를 위해 설계된 기법들입니다.

### Cutout / Random Erasing

이미지의 무작위 위치에 직사각형 영역을 0 또는 무작위 픽셀값으로 가립니다. 모델이 객체의 일부가 가려진 상황에서도 인식할 수 있도록 학습시킵니다. 실제 환경에서 객체가 다른 물체에 의해 부분적으로 가려지는 상황을 시뮬레이션합니다.

### Mixup

두 개의 이미지와 레이블을 선형 보간해 새로운 이미지를 만드는 방법입니다.

```
새 이미지 = λ × 이미지A + (1-λ) × 이미지B
새 레이블 = λ × 레이블A + (1-λ) × 레이블B
```

두 이미지가 반투명하게 겹쳐진 형태가 됩니다. 모델이 결정 경계를 더 부드럽게 학습하게 해 과적합을 줄이고 일반화 성능을 높입니다.

### CutMix

Mixup의 변형으로, 두 이미지를 픽셀 단위로 섞는 대신 한 이미지의 직사각형 영역을 다른 이미지에서 잘라온 영역으로 교체합니다. 레이블은 교체된 영역의 비율에 따라 혼합됩니다. Mixup보다 더 자연스러운 형태의 증강이며, ImageNet 분류에서 강한 성능 향상을 보여 많이 사용됩니다.

### AutoAugment / RandAugment

증강 정책을 사람이 직접 설계하는 대신 자동으로 최적화하는 방법입니다. AutoAugment는 강화 학습을 사용해 데이터셋에 가장 효과적인 증강 조합을 탐색합니다. RandAugment는 더 단순화된 방식으로 무작위로 선택한 증강 기법들을 적용합니다. 최신 이미지 분류 모델들이 이 방식을 기본 증강 전략으로 채택합니다.

---

## 7. 증강 적용 시 주의할 점

데이터 증강은 훈련 데이터에만 적용합니다. 검증 데이터와 테스트 데이터에는 적용하지 않습니다. 모델의 성능을 평가하는 단계에서 입력이 변형되면 평가 결과가 왜곡되기 때문입니다.

또한 도메인에 맞지 않는 증강은 오히려 성능을 떨어뜨릴 수 있습니다. 의료 영상에서 좌우 뒤집기는 적합하지만, 병리 조직 이미지에서 색상 변환이 과하면 임상적으로 의미 있는 색상 정보가 훼손될 수 있습니다. 텍스트 인식에서 과도한 회전은 문자를 읽을 수 없게 만듭니다. 증강의 종류와 강도는 항상 해결하려는 문제와 데이터의 특성을 고려해 선택해야 합니다.

## 핵심 요약

| 범주 | 기법 | 주요 효과 |
|---|---|---|
| **기하학적** | 뒤집기, 회전, 크롭, 이동, 전단 | 위치·방향 불변성 학습 |
| **색상** | 밝기, 대비, 채도, 색조, Color Jitter | 조명·색상 조건 다양화 |
| **노이즈·블러** | 가우시안 노이즈, 가우시안 블러 | 노이즈 강건성, 과적합 방지 |
| **고급** | Cutout, Mixup, CutMix, AutoAugment | 강력한 정규화, 결정 경계 부드럽게 |

데이터 증강은 데이터 부족 문제를 해결하는 동시에 과적합을 방지하고 일반화 성능을 높이는 핵심 전처리 기법입니다. 단순한 뒤집기나 크롭 같은 기법도 충분히 강력하지만, 문제의 성격과 도메인에 맞는 증강 전략을 설계하는 것이 성능 향상의 핵심입니다.
