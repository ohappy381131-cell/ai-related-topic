# 선형대수학과 머신러닝/딥러닝

## 선형대수학이란 무엇인가

선형대수학(Linear Algebra)은 **벡터(vector)** 와 **행렬(matrix)** 을 다루는 수학의 한 분야입니다. 더 정확히는, 벡터 공간(vector space)과 그 위에서의 선형 변환(linear transformation)을 연구하는 학문입니다. 핵심 질문은 이것입니다.

> "어떤 데이터를 공간 속의 점으로 표현했을 때, 그 공간을 어떻게 체계적으로 변환하고, 분석하고, 압축할 수 있는가?"

---

## 왜 머신러닝/딥러닝에 필수인가

### 1. 데이터 자체가 행렬이다

모든 데이터는 결국 숫자의 배열입니다.

- 이미지 1장 = 픽셀값들의 행렬 (예: $224 \times 224 \times 3$ 텐서)
- 텍스트 1문장 = 토큰 임베딩 벡터들의 집합
- 표 데이터 1개 = 행(샘플) × 열(피처)의 행렬

데이터를 다루는 것 자체가 행렬 연산입니다.

---

### 2. 뉴럴 네트워크의 순전파(Forward Pass)는 행렬 곱셈의 연속이다

하나의 레이어(layer)는 수식으로 이렇게 표현됩니다.

$$\text{출력} = f(W \cdot x + b)$$

여기서 $W$가 가중치 행렬(weight matrix), $x$가 입력 벡터, $b$가 편향(bias) 벡터입니다. 수천 개의 레이어가 쌓인 딥러닝 모델은, 본질적으로 **행렬 곱셈의 깊은 연쇄**입니다. GPU가 딥러닝을 빠르게 만드는 이유도, GPU가 행렬 곱셈에 특화된 병렬 처리 장치이기 때문입니다.

---

### 3. Attention 메커니즘은 선형대수의 정수다

GPT, BERT, Claude 같은 트랜스포머(Transformer) 모델의 핵심인 어텐션(Attention)은 다음 수식으로 동작합니다.

$$\text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^\top}{\sqrt{d}}\right) V$$

- $Q$(Query), $K$(Key), $V$(Value)가 모두 행렬
- $QK^\top$는 행렬 곱셈으로 토큰 간 유사도 계산
- $\sqrt{d}$는 차원(dimension) 수에 따른 스케일링
- $V$와의 최종 곱셈으로 문맥 정보 가중합

어텐션을 이해하려면 행렬 곱셈, 내적(dot product), 소프트맥스를 반드시 알아야 합니다.

---

### 4. 고유값 분해(Eigendecomposition)와 SVD는 차원 축소의 수학적 토대다

- **PCA(주성분 분석, Principal Component Analysis)**: 공분산 행렬의 고유벡터(eigenvector)들이 곧 주성분 방향입니다. 데이터의 분산이 가장 큰 방향을 찾아 불필요한 차원을 제거합니다.
- **SVD(특잇값 분해, Singular Value Decomposition)**: 임의의 행렬을 세 행렬의 곱으로 분해합니다. 추천 시스템의 협업 필터링(Collaborative Filtering)이 이 원리로 작동합니다.

---

### 5. 코사인 유사도는 내적의 기하학적 해석이다

벡터 검색(Vector Search)과 임베딩 유사도는 전부 내적(dot product)으로 계산됩니다.

$$\cos(\theta) = \frac{a \cdot b}{\|a\| \times \|b\|}$$

RAG(검색 증강 생성), 의미론적 검색, 벡터 데이터베이스 — 이 모든 것의 수학적 기반이 노름(norm)과 내적입니다.

---

### 6. 역전파(Backpropagation)는 행렬 미분이다

학습 과정에서 손실 함수(loss function)의 기울기를 계산하는 역전파는, 야코비안 행렬(Jacobian matrix)의 연쇄 법칙(chain rule) 적용입니다. 그래디언트(gradient) 자체가 벡터이고, 최적화 알고리즘인 Adam, SGD도 모두 행렬/벡터 연산입니다.

---

## 요약: 선형대수가 없으면 ML/DL은 없다

| 선형대수 개념 | ML/DL에서의 역할 |
|---|---|
| 벡터/행렬/텐서 | 데이터의 모든 표현 |
| 행렬 곱셈 | 뉴럴넷 순전파, Attention |
| 고유값/고유벡터 | PCA, 스펙트럼 분석 |
| SVD | 추천 시스템, 압축 |
| 내적·노름 | 유사도, 임베딩 검색 |
| 야코비안·헤시안 | 역전파, 2차 최적화 |

선형대수학은 ML/DL의 언어입니다. 논문을 읽고, 모델을 설계하고, 결과를 해석하는 모든 과정에서 이 언어를 모르면 표면만 다루게 됩니다.
