# 13. CNN(Convolutional Neural Network) — 각 레이어의 역할

본 글에서는 CNN을 구성하는 각 레이어가 어떤 역할을 하는지를 하나의 흐름 속에서 정리합니다. 왜 일반 신경망으로는 이미지를 처리하기 어려운지부터 시작해, 합성곱 레이어부터 출력 레이어까지 각 레이어가 데이터를 어떻게 변환하고 무엇을 학습하는지를 구조적으로 이해할 수 있도록 설명합니다.

## 1. CNN이란: 왜 일반 신경망으로는 부족한가

이미지를 일반적인 완전 연결 신경망(Fully Connected Network)으로 처리하면 어떻게 될까요? 예를 들어 28×28 픽셀 이미지 하나는 784개의 픽셀값으로 이루어져 있습니다. 이것을 입력층에 그대로 펼쳐 넣으면 784개의 뉴런이 필요합니다. 이미지가 커질수록 파라미터 수가 폭발적으로 늘어납니다. 또한 완전 연결 구조는 픽셀들 사이의 공간적 관계(인접한 픽셀이 관련이 있다는 것)를 전혀 활용하지 못합니다. 이미지를 뒤섞어놓아도 같은 방식으로 처리됩니다.

CNN은 이 두 가지 문제를 해결하기 위해 설계되었습니다. 이미지의 공간적 구조를 보존하면서 처리하고, 파라미터를 효율적으로 공유해 계산 비용을 줄입니다. 이를 가능하게 하는 것이 CNN의 독특한 레이어 구조입니다.

---

## 2. 합성곱 레이어 (Convolutional Layer)

합성곱 레이어는 CNN의 핵심이자 이름의 유래입니다. 이 레이어가 하는 일은 **이미지에서 특성(feature)을 추출**하는 것입니다.

핵심 개념은 필터(filter) 또는 커널(kernel)입니다. 필터는 작은 크기의 가중치 행렬입니다. 예를 들어 3×3 크기의 필터가 이미지 위를 왼쪽에서 오른쪽으로, 위에서 아래로 슬라이딩하며 이미지의 각 위치와 점곱(dot product) 연산을 수행합니다. 이 과정을 합성곱(convolution) 연산이라고 합니다. 그 결과로 특성 맵(feature map)이 생성됩니다.

```
이미지(5×5)        필터(3×3)         특성 맵(3×3)
┌─────────────┐    ┌─────────┐       ┌─────────┐
│ 1  2  3  0  1│   │ 1  0 -1│       │         │
│ 0  1  2  3  0│ × │ 1  0 -1│   =   │   ...   │
│ 1  0  1  2  3│   │ 1  0 -1│       │         │
│ 0  1  0  1  2│   └─────────┘       └─────────┘
│ 1  2  1  0  1│
└─────────────┘
```

필터의 가중치가 학습되면서 특정 패턴을 감지하는 능력이 생깁니다. 초반 레이어의 필터는 선, 모서리, 색상 변화 같은 단순한 패턴을 감지하고, 레이어가 깊어질수록 눈, 코, 귀처럼 복잡한 패턴을 감지하는 필터가 형성됩니다.

합성곱 레이어의 중요한 특성 두 가지가 있습니다. 첫째는 **가중치 공유(weight sharing)**입니다. 하나의 필터가 이미지 전체를 슬라이딩하며 같은 가중치를 사용합니다. 이미지의 어느 위치에 있는 고양이 귀든 같은 필터가 탐지합니다. 이 덕분에 파라미터 수가 대폭 줄어듭니다. 둘째는 **평행이동 불변성(translation invariance)**입니다. 객체가 이미지의 어느 위치에 있어도 감지할 수 있습니다.

합성곱 레이어의 주요 하이퍼파라미터는 필터의 크기(kernel size), 필터의 수(number of filters), 스트라이드(stride, 필터가 한 번에 이동하는 픽셀 수), 패딩(padding, 이미지 경계에 0을 추가해 출력 크기를 유지하는 기법)입니다.

---

## 3. 활성화 함수 레이어 (Activation Layer): ReLU

합성곱 레이어의 출력에는 활성화 함수가 적용됩니다. CNN에서는 ReLU(Rectified Linear Unit)가 가장 널리 사용됩니다.

```
ReLU(x) = max(0, x)
```

ReLU는 음수 값을 모두 0으로 만들고 양수 값은 그대로 통과시킵니다. 이 단순한 연산이 두 가지 중요한 역할을 합니다. 첫째, 비선형성을 도입합니다. 합성곱 연산은 선형 연산이므로, ReLU 없이 레이어를 아무리 쌓아도 결국 하나의 선형 변환과 동일합니다. ReLU가 있어야 깊은 네트워크가 복잡한 패턴을 표현할 수 있습니다. 둘째, 기울기 소실(vanishing gradient) 문제를 완화합니다. Sigmoid나 tanh와 달리 ReLU는 양수 영역에서 기울기가 1로 일정하므로, 역전파 시 기울기가 소실되지 않고 깊은 레이어까지 전달됩니다.

---

## 4. 풀링 레이어 (Pooling Layer)

풀링 레이어는 특성 맵의 공간적 크기를 줄이는 역할을 합니다. 이를 다운샘플링(downsampling) 또는 서브샘플링(subsampling)이라고 합니다.

가장 널리 사용되는 것은 **최대 풀링(Max Pooling)**입니다. 특성 맵을 일정 크기(예: 2×2)의 윈도우로 나누고, 각 윈도우에서 최댓값만 선택합니다.

```
특성 맵(4×4)           Max Pooling(2×2)     결과(2×2)
┌────────────────┐                          ┌──────────┐
│  1   3   2   4 │      윈도우마다           │  6    4  │
│  5   6   1   2 │   → 최댓값 선택    →     │  8    6  │
│  3   8   2   1 │                          └──────────┘
│  4   2   5   6 │
└────────────────┘
```

풀링 레이어가 하는 역할은 세 가지입니다. 첫째, 특성 맵의 크기를 줄여 이후 레이어의 계산량과 파라미터 수를 줄입니다. 둘째, 위치 변화에 대한 강건성을 높입니다. 객체가 조금 이동하거나 회전해도 최댓값은 크게 변하지 않습니다. 셋째, 과적합을 어느 정도 방지합니다.

평균 풀링(Average Pooling)은 최댓값 대신 평균값을 사용합니다. 최근에는 풀링 레이어 대신 스트라이드가 2인 합성곱 레이어로 대체하는 경우도 늘고 있습니다.

---

## 5. 배치 정규화 레이어 (Batch Normalization Layer)

배치 정규화는 각 레이어의 출력을 미니배치 단위로 정규화하는 레이어입니다. 합성곱 레이어와 활성화 함수 사이에 주로 배치됩니다.

학습이 진행되면서 각 레이어의 입력 분포가 계속 변하는 현상을 내부 공변량 이동(Internal Covariate Shift)이라고 합니다. 이전 레이어의 파라미터가 바뀌면 다음 레이어가 받는 입력의 분포도 바뀌는데, 이것이 학습을 불안정하게 만들고 학습률을 크게 설정하기 어렵게 합니다.

배치 정규화는 각 레이어의 출력을 평균 0, 분산 1로 정규화해 이 문제를 해결합니다. 덕분에 더 높은 학습률을 사용할 수 있고, 학습이 빠르게 수렴하며, 드롭아웃 없이도 어느 정도 과적합을 방지하는 효과가 있습니다.

---

## 6. 완전 연결 레이어 (Fully Connected Layer)

합성곱, 풀링 레이어를 거친 특성 맵은 공간적 구조를 가진 3차원 형태입니다. 완전 연결 레이어에 입력하기 전에 이것을 1차원 벡터로 펼치는 플래튼(Flatten) 과정이 필요합니다.

완전 연결 레이어는 일반 신경망의 레이어와 동일합니다. 모든 입력 뉴런이 모든 출력 뉴런과 연결됩니다. CNN에서 이 레이어의 역할은 앞선 레이어들이 추출한 특성들을 종합해 최종 분류 결정을 내리는 것입니다. 합성곱 레이어가 "무엇이 있는가"를 탐지한다면, 완전 연결 레이어는 "그것이 어떤 클래스인가"를 판단합니다.

---

## 7. 출력 레이어 (Output Layer)

마지막 완전 연결 레이어의 출력에 Softmax 또는 Sigmoid 활성화 함수를 적용해 최종 예측 확률을 출력합니다. 다중 클래스 분류에는 Softmax를, 이진 분류에는 Sigmoid를 사용합니다.

---

## 8. CNN의 전체 흐름

```
입력 이미지
    ↓
[합성곱 레이어] → 특성 추출 (선, 모서리 등 단순 패턴)
    ↓
[배치 정규화]  → 출력 분포 안정화
    ↓
[ReLU]         → 비선형성 도입
    ↓
[풀링 레이어]  → 크기 축소, 위치 불변성
    ↓
   ... (위 블록 반복, 깊어질수록 복잡한 패턴 학습)
    ↓
[Flatten]      → 3D → 1D 변환
    ↓
[완전 연결 레이어] → 특성 종합, 분류 결정
    ↓
[Softmax/Sigmoid]  → 클래스별 확률 출력
    ↓
최종 예측
```

레이어가 깊어질수록 추출되는 특성이 점점 추상화됩니다. 초반 레이어는 선과 모서리 같은 저수준 특성을, 중간 레이어는 질감이나 부분적 형태를, 후반 레이어는 눈, 코, 얼굴 전체 같은 고수준 특성을 학습합니다. 이 계층적 특성 학습이 CNN이 이미지 인식에서 압도적인 성능을 보이는 핵심 이유입니다.

---

## 핵심 요약

| 레이어 | 역할 |
|---|---|
| **합성곱 레이어** | 필터로 이미지에서 특성 추출, 가중치 공유로 파라미터 절약 |
| **ReLU** | 비선형성 도입, 기울기 소실 완화 |
| **풀링 레이어** | 특성 맵 크기 축소, 위치 불변성 확보, 계산량 감소 |
| **배치 정규화** | 출력 분포 안정화, 학습 속도 향상, 과적합 방지 |
| **완전 연결 레이어** | 추출된 특성 종합해 최종 분류 결정 |
| **출력 레이어** | 클래스별 예측 확률 출력 |
